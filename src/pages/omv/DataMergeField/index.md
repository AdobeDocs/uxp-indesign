# DataMergeField
A data merge field.

## Instance

> 
> String **fieldName** 
>
> The name of the field.
*** 
> *Read Only* 
> 
> SourceFieldType **fieldType** 
>
> The source field type.
*** 
> *Read Only* 
> 
> Boolean **isValid** 
>
> Returns true if the object specifier resolves to valid objects.
*** 
> *Read Only* 
> 
> DataMerge **parent** 
>
> The parent of the DataMergeField (a DataMerge).
*** 
> *Read Only* 
> 
> Number **index** 
>
> The index of the DataMergeField within its containing object.
*** 
> *Read Only* 
> 
> Events **events** 
>
> A collection of events.
*** 
> *Read Only* 
> 
> EventListeners **eventListeners**

*** 
> Object **properties** 
>

> String **toSource**()
> 
> Generates a string which, if executed, will return the DataMergeField.
*** 
> DataMergeField **getElements**()
> 
> Resolves the object specifier, creating an array of object references.
***

> 
> Retrieves the object specifier.
***
