# AdornmentOverprint
Overprint options for kenten marks.

## Class
> *Read Only* 
> 
> Number **AUTO** = 1635019116
> 
> Uses auto overprint.
*** 
> *Read Only* 
> 
> Number **OVERPRINT_ON** = 1701736302
> 
> Turns on overprint.
*** 
> *Read Only* 
> 
> Number **OVERPRINT_OFF** = 1701736294
> 
> Turns off overprint.
