# SaveOptions
Options for saving a document before closing or quitting.

## Class
> *Read Only* 
> 
> Number **NO** = 1852776480
> 
> Does not save changes.
*** 
> *Read Only* 
> 
> Number **ASK** = 1634954016
> 
> Displays a prompts asking whether to save changes.
*** 
> *Read Only* 
> 
> Number **YES** = 2036691744
> 
> Saves changes.
