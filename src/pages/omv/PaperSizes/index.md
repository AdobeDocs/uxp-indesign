# PaperSizes
Paper size optons.

## Class
> *Read Only* 
> 
> Number **DEFINED_BY_DRIVER** = 1347634290
> 
> Allows the printer driver to define the paper size.
*** 
> *Read Only* 
> 
> Number **CUSTOM** = 1131639917
> 
> Allows definition of a custom paper size. Note: Not all printers allow custom paper sizes.
