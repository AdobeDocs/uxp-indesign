# XMLRuleMatchData
An XML element matched by a rule or rules in the current XML rule set.

## Instance

> An XML element matched by a rule or rules in the current XML rule set.
*** 
> *Read Only*

*** 
> *Read Only*

*** 
> *Read Only*
