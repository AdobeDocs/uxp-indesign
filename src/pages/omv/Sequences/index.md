# Sequences
Options for printing page sequences.

## Class
> *Read Only* 
> 
> Number **ALL** = 1634495520
> 
> Prints all pages.
*** 
> *Read Only* 
> 
> Number **ODD** = 1868850208
> 
> Prints only odd-numbered pages.
*** 
> *Read Only* 
> 
> Number **EVEN** = 1702258030
> 
> Prints only even-numbered pages.
